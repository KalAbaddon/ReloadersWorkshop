NOTICE:

This is the SAAMI PDF File Folder

SAAMI PDF Files downloaded by Reloader's WorkShop will be placed in this folder.  

Once a specific SAAMI PDF has been downloaded, Reloader's WorkShop will display the file contained in this folder rather than download it again.

If you feel that SAAMI has updated a specific drawing, simply delete it from this folder and Reloader's WorkShop will download the latest version the next time you request it.
